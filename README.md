# Project setup
```js
git clone https://github.com/imadys/jadwal-frontend.git
cd jadwal-frontend
```
copy .env.example and rename it to .env.local
```js
npm install
npm run dev
```

```js
NEXT_PUBLIC_BACKEND_URL=http://localhost:8000
```
paste your backend link
